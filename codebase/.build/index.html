THIS SEMESTER'S PROJECTS
Ongo
IoT
Ongo is a startup helping health and fitness experts create mobile learning experiences. Its app provides users with interactive courses and tutorials that integrate with other devices, creating a personalized data-driven experience. This semester, we're working to integrate Ongo's platform with various devices, such as Amazon Echo.
Sutro
App Development
Based out of San Francisco, Sutro is a hardware startup developing a smart pool monitor that measures pool chemistry and delivers the appropriate pool chemicals as they're depleted. CodeBase is working with Sutro to develop a cross-platform mobile application that allows users to monitor their pool chemistry, view pool analytics, and easily order materials.
Crowdbotics
NLP/AI Chatbot
Crowdbotics is a startup automating the process of connecting clients with developing needs with capable engineers. We're working with Crowdbotics to develop a chatbot that interviews potential engineers for a project and applies NLP algorithms to choose which engineers are the right fit for a software development project.
